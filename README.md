
# Full Stack App + CRUD

Aplicação web (frontend e backend) com um CRUD simples e dockerizada.




## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/fomes/book-store
```

Entre no diretório do projeto

```bash
  cd book-store
```

Inicie o servidor

```bash
  docker compose up
```

## Stack utilizada

**Front-end:** Bootstrap

**Back-end:** Spring Boot, Thymeleaf, Postgres.
